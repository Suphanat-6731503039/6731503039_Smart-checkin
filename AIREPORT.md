# AI Usage Report

## AI Tools Used

The following AI tools were used during the development of this project:

* ChatGPT (OpenAI)

## How AI Was Used

AI was used as a development assistant to help generate initial code structures and technical guidance during the implementation of the Smart Class Check-in & Learning Reflection App.

Specifically, AI assisted with:

* Generating the initial **Flutter project structure**
* Providing example code for **QR code scanning integration**
* Suggesting implementation methods for **GPS location retrieval**
* Creating sample code for **Firebase Firestore data storage**
* Drafting the **Product Requirement Document (PRD)**
* Generating template code for UI components such as **forms and navigation**

## Student Contributions

The student reviewed, modified, and integrated the generated code into a working application. The following tasks were implemented or customized by the student:

* Adjusting the **Flutter UI layout and navigation**
* Integrating **Firebase configuration**
* Modifying the **form input fields and validation logic**
* Organizing the project structure for maintainability
* Testing application features including check-in and finish class workflows
* Deploying the web version using **Firebase Hosting**

## Reflection

AI was used as a supporting tool to speed up development and provide coding references. However, the student was responsible for understanding the generated code, integrating different components, and ensuring the application works as intended.

The final system represents the student's implementation and understanding of mobile application development using Flutter and Firebase.
