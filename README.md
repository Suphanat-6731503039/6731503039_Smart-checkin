# Smart Class Check-in & Learning Reflection App

Flutter prototype app for university attendance and learning reflection.

## Features
- Student Login (Firebase Auth)
- Class Check-in with QR
- GPS classroom validation
- Mood reflection emoji selector
- Finish class reflection
- Instructor dashboard
- Attendance analytics chart
- Firebase Firestore storage

## Setup

1. Install Flutter
2. Install Firebase CLI
3. Create Firebase project
4. Run:

flutter pub get
flutter run

## Build Web

flutter build web

## Deploy

firebase init hosting
firebase deploy
